**Control de Caja**

Aplicacion de escritorio para el control de caja por empresa, con una intefaz intuitiva tipo Excel.


🚀 **Características actuales**
 - Registro de ingresos y egresos
 - Cálculo automático de saldo
 - Totales automáticos en la parte inferior
 - Interfaz amigable tipo hoja de cálculo
 - Control por empresas
 - Diseño limpio y enfocado en productividad

🛠️ **Tecnologías utilizadas**
 - Python
 - PySide6 (Qt)
 - Qt Widgets
 - Git & GitHub

🎯 **Objetivo**
 - Simplicidad similar a excel
 - interfaz moderna e intuitiva
